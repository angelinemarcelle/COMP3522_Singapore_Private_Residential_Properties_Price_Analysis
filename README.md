## Overview

This project provides an in-depth analysis of Singapore's private residential property prices using a combination of Jupyter Notebooks and Python scripts. The aim is to uncover trends, patterns, and insights in the property market, assisting investors, homeowners, and analysts in making data-driven decisions.

## Features

- **Data Exploration:** Clean and preprocess property price datasets for analysis.
- **Visualization:** Generate interactive charts and graphs to visualize price trends, volume, and other market factors.
- **Statistical Analysis:** Apply statistical methods to reveal significant patterns and correlations.
- **Machine Learning (Optional):** Includes models for price prediction and market segmentation (if applicable).
- **Jupyter Notebooks:** Step-by-step analysis and documentation, ideal for reproducibility and collaboration.

## Repository Structure


- Singapore government property price datasets (HDB, URA, etc.)

- Run notebooks to follow and reproduce analysis.
- Use Python scripts for custom processing or extension.
